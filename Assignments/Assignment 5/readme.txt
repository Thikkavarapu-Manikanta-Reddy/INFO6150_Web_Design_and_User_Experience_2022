Purpose:
Splitter is a web application that helps in keeping track of your shared expenses and balances with trips,friends,groups and family.

Bootstrap Components Used:

1. Nav: It is used for creating navigation bar.
2. button: It is used for adding a login button which opens up a login page
3. grid: It is used across the page
4. container:Containers are the most basic layout element in Bootstrap and are required when using our default grid system.
5. footer: A footer is a navigation method for websites. It holds links, buttons, company info, copyrights, forms and many other important elements.
6. modal:- Modal is used for login popup
7. carousel:It is used for displaying our products
8. form: It is used for creating a login form 
9. spinner:It is used before page loads 
10.alert : Alerts are available for any length of text, as well as an optional dismiss button. For proper styling, use one of the eight required contextual classes (e.g., .alert-warning).
11.tooltip :It is used for icons
12.cards : It is used it to hold images in  testimonals
